AI Avatar Widget - Minimal Integration

This page demonstrates the simplest way to integrate the AI Avatar Widget using data attributes for configuration.

Key Benefits:

  • Fully customizable through data attributes
  • No JavaScript coding required
  • Automatically loads all dependencies

Integration Code

Just add these two lines to your website:

<!-- Widget container with your configuration -->
<div id="ai-avatar-widget" 
     data-knowledge-id="your_knowledge_id"
     data-avatar-name="Tyler-insuit-20220721"
     data-opening-message="Hello! I'm Aiva. How can I help you today?"></div>

<!-- Widget loader script -->
<script src="./assets/avatar-widget-loader.js"></script>

Available Configuration Options

  • data-knowledge-id - Your knowledge base ID (required)
  • data-avatar-name - Avatar model to use
  • data-opening-message - First message shown to users
  • data-position - Position on screen (bottom-right, top-left, etc.)
  • data-theme - Color theme (light, dark)
